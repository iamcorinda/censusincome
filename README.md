# Census Income SQL Analysis

## Project Overview
This project explores the UCI Adult Census Income dataset to analyze 
the relationship between education level and income using SQL.

## Dataset
- **Source:** Kaggle - Adult Census Income
- **Description:** Census data containing demographic information 
including age, education, occupation, and income level
- **Records:** ~32,000 rows

## Questions Explored
- Which education levels produce the most high earners?
- What percentage of each education group earns more than $50K?

## Key Findings
Doctorate and Prof-school degree holders have the highest probability 
of earning more than $50K at 74.1% and 73.4% respectively, 
compared to only 16% of high school graduates.
